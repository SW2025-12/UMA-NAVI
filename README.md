# UMA-NAVI

## 🐴 システム概要
競馬ファンが現在や過去の馬の戦績や出走データなどを簡単に検索し、推し馬を継続して追いかけられる、初心者にも見やすい競馬情報管理システムを提供します。

## 🎯 開発目的
* 推し馬を継続して追いかけられるようにする。
* 初心者にも見やすい競馬情報管理システムを提供する。

## 🛠️ 機能一覧
1. 認証 / ログイン
2. 馬のデータ閲覧（プロフィール・血統・戦績）
3. 検索機能（馬名）
4. お気に入り機能（追加・削除中心）
5. レース結果閲覧
